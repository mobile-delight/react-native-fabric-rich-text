# react-native-fabric-html-text

Fabric-first HTML text renderer for React Native

## Installation


```sh
npm install react-native-fabric-html-text
```


## Usage


```js
import { FabricHtmlTextView } from "react-native-fabric-html-text";

// ...

<FabricHtmlTextView color="tomato" />
```
